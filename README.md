"# Mid-Assignment"

## Overview

This is a library management system with user and admin functionality.

## Getting Started

### Prerequisites

- .NET 8 SDK
- Node.js (v18+)
- SQL Server

### Backend Setup

```bash
cd backend
dotnet restore
dotnet run
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
